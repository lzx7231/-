小结

从以上图解可以看出，对已初始化过的 基本类型变量 进行赋值操作，值会被覆写，类型也有可能会改变。例如：

var a = 1; // 此时 a 类型为 Number，值为 1
a = 'Hello'; // 重新赋值，此时 a 类型为 String，值也变了。

将已初始化赋值过的 引用类型变量 直接赋值给 另一变量，只是在栈内存中拷贝了一份同样的指针，指向同一个堆内存。
